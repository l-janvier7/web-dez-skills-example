WEB DEVELPOMENT SKILLS

Learn HTML

HMTL stands for Hyper Text Markup Language. HTML is the content of a web page.
Emaples of HMTL include headings, paragraphs, images, links, lists,etc.

Check out the HTML Tutorial on w3schools.com.

Learn CSS

CSS stands for Cascading Stlye Sheets. CSS is the style of a web page. Examples of CSS include
font styles and colour, backround colour, borders, dimensions, layout, etc.

Check out the CSS Tutorial on w3schools.com.

Learn JavaScript

JavaScript is the programing language of the web. JavaScript helps to make web pages interactive and dynamic.
Some common JavaScript tasks are:

  • -Add new HTML to the page,
  • -Change the existing content / style,
  • -React to the user events (mouse clicks, key presses, etc.)

Check out the Javascript Tutorial on javascript.info.
